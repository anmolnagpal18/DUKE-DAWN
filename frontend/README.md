# Hoodie Store Frontend

React.js frontend for the premium hoodie store.

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

The app will run on http://localhost:3000

## Build

```bash
npm run build
```

## Environment Variables

Create a `.env` file in the root directory:

```
REACT_APP_API_URL=http://localhost:5000/api
```

## Features

- Responsive design (mobile, tablet, desktop)
- Dark theme with gold accents
- Product browsing and filtering
- Shopping cart and wishlist
- User authentication
- Order management
- Checkout flow

## Technology Stack

- React.js 18
- React Router v6
- Tailwind CSS
- Axios for API calls
